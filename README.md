# Features
   - Splash Screen
   - Onboarding Screen
   - News from multi Language
   - Top News
   - All News
   - News by categories
   - -Search News
   - News Details
   - Native WebView for news source

# How to use ?
   - Clone thhe repo
   - Go to  https://www.thenewsapi.com/ and create a free account
   - Go to Dashboard to get you api token
   - Paste the token in Constant File
   - Done!
